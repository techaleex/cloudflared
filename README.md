# cloudflared

<h1>How to Install Cloudflared</h1>

- $ ` apt update && apt upgrade `
- $ ` apt install git `
- $ ` apt install php `
- $ ` git clone https://github.com/techaleex/cloudflared `
- $ ` cd cloudflared `
- $ ` chomd 777 setup `
- $ ` ./setup `

<p>Type cloud then tool Run</p>
- $ `cloud` </br>
- $ ` cd cloudflared `
- $ ` chomd 777 setup `
- $ ` ./setup `
